# 科学数据分析（通用方法论）

仪器特定的判读知识（如色谱判峰）在对应仪器/装置技能里；本技能只管跨装置通用的方法论。

## 三分法纪律

1. **确定性计算给脚本，不心算**：统计量、拟合、单位换算——写一段脚本（Python/JS）
   用工作台执行能力跑出来，把脚本与输出一并展示。批量/复用价值高的脚本以
   `artifact_save(kind=tool)` 登记沉淀。
2. **语义判读给你（LLM）**：趋势含义、异常解释、与假设的关系——给判断也给依据。
3. **领域解读查领域技能**：涉及特定仪器数据格式与判读要点时，引用对应仪器技能。

## 方法论清单（按需取用）

- 描述统计先行：n、均值/中位数、分散度、缺失值——任何结论前先看数据长相。
- 对照检查：有无对照组/空白/重复？没有就在结论里明说局限。
- 异常值：标记而非默默剔除；剔除须经用户同意并记录理由。
- 拟合：报告参数置信区间与拟合优度，不只报曲线。
- 多重比较、样本量过小、相关当因果——三个常见坑，主动提示。

## 产物纪律

- 分析结论 `artifact_save(kind=analysis)`，溯源引用数据来源
  （`scicompass://results/<id>` / 文献 / 图谱节点）。
- 阴性结论同等登记——失败经验是组图最值钱的晋升候选。
- 结论与组图先验或 SciGraph 公共知识冲突时：明确标注冲突双方与证据强度，
  提请科学家裁决（要么错了，要么是发现）。
